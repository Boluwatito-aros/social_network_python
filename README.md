# social_network_python

A console-based social network simulator built in Python.

## About
Designed and built a console-based social network simulator in Python, modelling user profiles,
friend connections, and activity feeds using graph data structures (adjacency lists).

## Features
- Add/remove friends and view mutual connections
- Suggest new friends based on second-degree relationships, mirroring real-world social graph traversal
- Breadth-First Search (BFS) to compute degrees of separation between users
- Structured using OOP principles ensuring scalability and clean separation of concerns

## Tech
Python · Graph data structures (adjacency lists) · BFS · OOP
